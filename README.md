# taskleaf
Ruby on Railsを使用して作成したタスク管理アプリ。タスクの作成/閲覧/編集/削除が可能。検索機能やインポート、エクスポート機能もあります。

## 環境
- Ruby:2.4.1
- Rails:5.2.3

## 基本機能
- タスクの作成/閲覧/編集/削除
- タスク名による検索機能
- タスクのインポート、エクスポート(CSV形式)
- ログイン機能(管理者権限ありのユーザーによって管理されている)
